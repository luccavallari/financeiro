* [Allan Gonçalves da Cruz](https://github.com/Allangcruz) - Desenvolvedor web

## Tecnologias utilizadas

### Backend
* Codeigniter - 3.0.4
* Mysql - 5.5.44
* Apache - 2.4.7

### Frontend
* Jquery - 2.2.0
* Bootstrap - 3.3.6
* jquery-validate - 1.14.0
